# Portfolio Site Brief

## 参考サイト調査サマリー

### brittanychiang.com
- **構造**: 左カラム固定（名前・肩書き・ナビ・SNSリンク）＋右カラムスクロール（About / Experience / Projects）の2カラムスプリットレイアウト。
- **カード構造**: Experience / Projects は横長カード。左に年度や画像、右にタイトル・説明・技術タグ（ピル型）。ホバーで背景がほのかに発光しタイトル色が変化。
- **ナビ**: 左カラムの縦並びアンカーリンク。現在地セクションがハイライトされる。
- **カラー**: ダークネイビー背景（#0a192f系）＋ティール/シアンのアクセント＋スレートグレーの本文。1アクセントカラーに絞った落ち着いた配色。
- **適用ポイント**: 左固定プロフィール＋右プロジェクト一覧の型は今回のAI開発ポートフォリオに最適。技術タグのピル表示を踏襲する。

### leerob.io
- **構造**: 上部ヘッダーナビ＋中央寄せ単一カラム。ヒーロー（自己紹介）→ 主要プロジェクト/リンク → ブログ一覧の縦積み。
- **カード構造**: シンプルなリスト寄りカード。タイトル＋短い説明＋メタ情報。余白を大きく取りミニマル。
- **ナビ**: 上部の水平ナビ（Home / About / Blog など）。
- **カラー**: ほぼモノクロ（白黒グレー）＋わずかなアクセント。タイポグラフィ主導のミニマルデザイン。
- **適用ポイント**: ヒーローセクションでの簡潔な自己紹介と、過剰な装飾を避けたカード設計を参考にする。

### joshwcomeau.com
- **構造**: 中央寄せカラム。明るくカラフルでプレイフル。ヒーロー → コンテンツカードのグリッド。
- **カード構造**: 角丸・影・グラデーションを使ったリッチなカード。アイコンやイラストを伴うことが多く、ホバーアニメーションが豊か。
- **ナビ**: 上部水平ナビ。
- **カラー**: 明るい紫/ピンク/青のグラデーション、ライト/ダークテーマ切替対応。多色だが調和。
- **適用ポイント**: カテゴリごとの色分けバッジ、角丸＋ホバーアニメーションのカード演出を取り入れる。

### 総合方針
brittanychiang の左固定プロフィール＋右プロジェクト一覧を骨格とし、joshwcomeau のカテゴリ色分けバッジとカードホバー演出、leerob のミニマルな自己紹介を組み合わせる。ダークテーマ＋1〜2アクセントカラー、技術タグはピル型、カテゴリは色分けバッジで表現する。

## 推奨サイト構造

1. **Header / Nav** — 名前・肩書き（AI開発者）、セクションアンカー（About / Projects / Contact）、SNS/GitHubリンク。デスクトップは左固定サイドバー、モバイルは上部に折りたたみ。
2. **Hero / About** — 簡潔な自己紹介（AIエージェント・自動化・スクレイピング・画像生成を扱う開発者であること）。
3. **Filter Bar** — カテゴリフィルター（All / Image / Agent / Automation / Scraping）。
4. **Projects Grid** — 8件のプロジェクトカードをレスポンシブグリッド（デスクトップ2〜3列、モバイル1列）で表示。フィルター連動。
5. **Contact / Footer** — メールアドレス、GitHub、コピーライト。
